getAttribute()

Used to retrieve the value of an attribute from an element.
element.getAttribute("href");
Get link targets (href)
Access custom data attributes

Flexbox

The display: flex property is used to create a flex container, allowing child elements to be arranged in a flexible layout. Default direction: row (horizontal), Enables easy alignment and spacing, Supports responsive design.

justify-content → horizontal alignment
align-items → vertical alignment
flex-direction → row / column

list-style

These properties are used to remove default styling from lists. Commonly used in navigation menus.
list-style: none; → removes bullets or numbers
padding: 0; → removes default spacing

querySelectorAll()

This JavaScript method selects all elements that match a CSS selector.
document.querySelectorAll("selector");
Returns a NodeList (collection of elements)
Supports CSS selectors (class, id, tag)
Often used with loops like forEach()

forEach()

The forEach() method is used to loop through each element in a collection.
Perform an action on every selected element
Commonly used with querySelectorAll()

addEventListener()

Used to attach an event (like click) to an element.
element.addEventListener("click", function);
Makes web pages interactive
Responds to user actions

preventDefault()

Stops the browser’s default behavior.
Prevent link navigation
Prevent form submission
